1. 초기 비용과 운영 비용을 면밀히 고려해보고 도입해야한다.  
2. 추후 마이그레이션이 필요한지... 필요할 것 같다면 그 부분도 생각해보기.
3. 성능 최적화 => 비용 절감 효과, 사용자에게 만족스러운 경험 제공
